## Instrucciones de instalacion y configuracion

Clone este repositorio. Necesitaras `node` y `npm` instalados globalmente en tu equipo.

## Instalacion:

Instalar dependencia 

`npm install`  

Inicia la Fake API

`json-server db.json`  

Inicia el servidor

`npm run serve` 

Visita el proyecto:

`localhost:8080`
